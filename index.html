INI File - Ringkasan Format Konfigurasi

INI file adalah jenis file konfigurasi sederhana yang digunakan oleh banyak sistem operasi dan aplikasi. Format ini biasanya terdiri dari section (bagian), properties (kunci dan nilai), serta komentar.

Struktur Umum

File INI biasanya memiliki struktur seperti ini:

[section_name]
key1=value1
key2=value2
; this is a comment

Sejarah

Format INI banyak digunakan pada sistem Microsoft Windows, terutama versi Windows 3.x dan Windows 9x, sebelum kemudian digantikan secara bertahap oleh Registry dan format XML/JSON.

Dukungan Bahasa Pemrograman

Banyak bahasa pemrograman seperti Python, PHP, dan C++ menyediakan parser bawaan atau library tambahan untuk membaca file INI.
